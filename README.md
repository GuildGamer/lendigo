# LENDIGO!

*Lendigo backend api for*:
- retrieving the most recent news using [HackerNews](https://news.ycombinator.com/) API

*Documentation*:
- [Documetation](https://documenter.getpostman.com/view/16607098/UyxgJo8n)

**TO START SERVER:**
chmod a+x start.sh
bash start.sh

**TO STOP SERVER:**
chmod a+x stop.sh
bash stop.sh

**GET LOGS RUN:** 
chmod a+x logs.sh
bash logs.sh
